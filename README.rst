============================
meridian-spectrometer-reader
============================
for Poker Flat Digital Meridian Spectrometer, which uses netCDF

`Data source <https://amisr.asf.alaska.edu/PKR/DMSP/NCDF/>`_

install
=======

    python setup.py develop
